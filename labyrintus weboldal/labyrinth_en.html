Labyrinth


Opening screen

There are three options on the opening screen. With the first two, you can choose whether it is single (egyjátékos), or we want to play in multiplayer (többjátékos) mode.
It is important that if you choose the multiplayer mode, you CANNOT use the game from one device! The third option allows you to set the language of the game.
The currently available languages ​​are Hungarian and English.


Single player and multiplayer modes

After choosing whether you want to play in single player or multiplayer mode, on the next screen you can choose which game mode you want to continue with.
In the time mode, a clock counts down and you have to find all the rooms in the labyrinth during this time.
When using the blind map mode, the map is not visible and you have to find all the rooms in this way.


Loading track

On the next screen, you can load already created tracks.
It is IMPORTANT that the name of the file containing the track must be entered together with the extension! (For example: sample.txt)
Then the game can begin!


The goal of the game

After filling the track, you have to find the rooms in the labyrinth by following the karat garden.
You can see the number of rooms and how many of them you have found in the upper left corner under the icon and the pedometer.
The game is over when you have found all the rooms and exit through the exit at the bottom of the track.
